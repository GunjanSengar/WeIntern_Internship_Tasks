# Student Management System

## Project Overview
This is a console-based Java application developed as part of the WeIntern Java Developer Internship Task.

The application helps manage student records using CRUD operations and demonstrates Core Java concepts such as:
- OOP (Object-Oriented Programming)
- ArrayList
- Enum
- Exception Handling
- Input Validation
- Scanner Class

---

# Features

✅ Add Student  
✅ View All Students  
✅ Search Student by ID  
✅ Update Student Details  
✅ Delete Student Records  
✅ Input Validation  
✅ Exception Handling  
✅ Menu Driven Console Application

## Step 1: Compile Java Files

```bash

```

```bash
java StudentManagementSystem
```
