# Student Management System

## Project Overview

This is a console-based Java application developed as part of the WeIntern Java Developer Internship Task.

The application allows users to manage student records using CRUD operations and demonstrates important Core Java concepts such as:

- Object-Oriented Programming (OOP)
- ArrayList Collection
- Enum
- Exception Handling
- Input Validation
- Scanner Class

---

# Features

✅ Add Student  
✅ View All Students  
✅ Search Student by ID  
✅ Update Student Details  
✅ Delete Student Records  
✅ Input Validation  
✅ Exception Handling  
✅ Menu-Driven Console Application

```bash

```

```bash
java StudentManagementSystem
```
