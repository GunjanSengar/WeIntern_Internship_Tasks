# Banking Application

## Project Overview

A console-based banking system developed using Java.

## Features

- Open Account
- Deposit Money
- Withdraw Money
- Check Balance
- Print Statement
- Transaction History
- Validation for Negative Amounts
- Insufficient Funds Check

## Technologies Used

- Java
- BigDecimal
- ArrayList
- OOP
- Exception Handling

## How to Run

```bash
javac *.java
java BankingApplication
```
